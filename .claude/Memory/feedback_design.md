## Websites and landing pages must look like a professional agency built them

### MUST have (non-negotiable)
- **Real images** — full-bleed, high-quality photography. NEVER placeholder divs, emojis, or colored boxes. Verify every image URL returns 200.
- **Full-screen hero** — background image filling viewport, dark overlay, text at bottom.
- **Visual depth** — parallax on images, hover zoom, gradient overlays, layered overlapping elements.
- **Cinematic sections** — alternating light/dark, image backgrounds on testimonials and CTAs.
- **Symmetry & rhythm** — consistent aspect ratios in grids. 50/50 splits must be balanced.

### Design language
- Muted earth tones, not saturated primaries
- Serif for headings (e.g., Playfair Display), clean sans for body
- Uppercase tracking (0.2-0.3em) for labels
- Warm accent color (gold, terracotta), not blue/purple
- Sharp edges for premium feel, not rounded-everything
- Custom scrollbar matching palette

### "Wow" elements
- Scrolling marquee banner between sections
- Parallax zoom-out on CTA section (reverse parallax)
- Giant decorative quote marks on testimonials
- Hover line accents on product cards
- Nav transitions from transparent to blurred on scroll
- Floating cards overlapping images

### NEVER
- Emojis as visual elements
- Flat colored boxes as image placeholders
- Generic 3-column card grid with rounded corners
- Inter, Roboto, or system fonts for headings
- Purple gradients
- Rounded buttons (use sharp rectangular with uppercase tracking)

### Quality gate before declaring done
1. All images verified (HTTP 200)
2. Build compiles (0 errors)
3. Skill rules followed (folder structure, footer, content separation)
4. Visual rhythm: no two adjacent sections look similar
5. Mobile: hero readable, images scale, CTAs tappable
